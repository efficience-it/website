Les bundles, ou « packages » en français, désignent « un regroupement de plusieurs fichiers ou logiciels » livrés comme un ensemble logique de fonctionnalités ou de composants. Leur format de package simplifie la distribution et l'installation des logiciels, permettant aux utilisateurs de télécharger et d'installer plusieurs composants simultanément via Composer.

## Les bundles natifs de Symfony

### HTTPClient

Introduit dans Symfony 4.3, HTTPClient permet une communication efficace avec des services externes via des requêtes HTTP et des API REST. Il offre des interfaces orientées objet pour envoyer des requêtes, gérer les réponses et manipuler les en-têtes, ainsi que des fonctionnalités incluant la gestion des cookies, la gestion des flux, le téléchargement de fichiers, la mise en cache et la gestion des exceptions.

### KnpPaginatorBundle

Ce bundle simplifie la pagination des résultats dans les applications Symfony en permettant une personnalisation en fonction des besoins spécifiques. Il fonctionne avec les résultats de requêtes Doctrine et d'autres sources de données, offrant une configuration flexible pour le nombre d'éléments par page, les pages de navigation et les paramètres d'URL.

### Mailer

Ce bundle crée et envoie des emails grâce à un système puissant. Les messages peuvent être structurés en plusieurs parties, créés à partir de templates avec mise en forme, inclure des pièces jointes et des en-têtes personnalisés. Il est entièrement testable au sein de l'écosystème Symfony.

## Les bundles souvent utilisés dans les projets d'Efficience IT

### Faker PHP

Génère des données fictives réalistes à des fins de test — remplissage de bases de données, création de documents de test ou anonymisation de données de production. Il prend en charge plusieurs langues grâce à des fonctionnalités de localisation.

### Snappy bundle

Convertit des documents HTML en fichiers PDF et autres formats comme PNG et JPEG, offrant des options de personnalisation et de configuration pour le contenu converti.

### Easy Admin

Simplifie la conception d'interfaces d'administration avec un minimum de code. Fournit des fonctionnalités CRUD (Create, Read, Update, Delete) pour la gestion des entités avec des interfaces intuitives et conviviales.

### Swagger Nelmio bundle

Génère automatiquement une documentation API interactive à l'aide de Swagger. Décrit, produit et consomme des applications RESTful. Nelmio gère indépendamment les en-têtes Cross-Origin Resource Sharing (CORS) et génère la documentation de l'API REST via des annotations et des spécifications Swagger.

### DataTable

Bundle entièrement modulable sans limitation de source de données. Utilise des adaptateurs lorsque des requêtes de données sont effectuées, renvoyant des ensembles de résultats avec des métadonnées sur le nombre d'enregistrements.

### LexikTranslation

Importe et exporte le contenu des fichiers de traduction dans des bases de données tout en fournissant des interfaces graphiques pour les modifications, permettant la vérification de traductions complètes dans tous les domaines.

### JWT Authentification

Fournit l'authentification par JSON Web Token, couramment utilisée pour sécuriser l'accès aux API. Permet l'authentification par token pour les API nécessitant une vérification des utilisateurs.

### VichUploaderBundle

Facilite les téléchargements de fichiers liés aux entités Doctrine ORM avec une personnalisation avancée. Utilise une configuration de « mapping » pour les entités prenant en charge les téléchargements de fichiers, réduisant le code complexe tout en offrant des limites de taille de fichier configurables, des formats autorisés et des noms générés automatiquement. Inclut des fonctionnalités de traitement d'images comme le redimensionnement, le recadrage et la conversion de format.

### PHP Office

Bundle open source pour la génération de documents bureautiques (Excel, Word, PowerPoint). Contient trois composants :

- **PhpWord** : crée et modifie des fichiers Word (.docx)
- **PhpSpreadsheet** : manipule des fichiers Excel
- **PhpPresentation** : crée des fichiers PowerPoint (.pptx)

Tous les composants sont activement maintenus par la communauté.

### Console bundle

Composant fondamental de Symfony fournissant une fonctionnalité d'interface en ligne de commande (CLI) pour l'interaction avec l'application, l'automatisation des tâches et l'exécution de scripts. Génère automatiquement la documentation d'aide pour les commandes.

### DoctrineBundle

Intègre les systèmes de gestion de bases de données et le mapping objet-relationnel via Doctrine dans les applications Symfony. Prend en charge la représentation ORM des données de base de données en tant qu'objets PHP, la gestion des migrations, la configuration des écouteurs d'événements et les paramètres de connexion à la base de données.

### TwigBundle

Moteur de templates moderne et puissant, automatiquement configuré lors de l'installation d'une application web Symfony. Permet des extensions et filtres personnalisés, gère les répertoires de templates, les options de cache et le système de mise en cache pour améliorer les performances dans les environnements de développement et de production.

### FOSUserBundle

Simplifie la gestion des utilisateurs avec des fonctionnalités prêtes à l'emploi pour l'authentification, l'autorisation, la gestion des utilisateurs et la réinitialisation de mot de passe, basées sur les bonnes pratiques Symfony. Fournit une configuration flexible pour créer, modifier, supprimer et rechercher des utilisateurs, ainsi que la gestion des rôles et des permissions.

### SensioFrameworkExtraBundle

Offre des fonctionnalités supplémentaires utilisant des annotations pour configurer le comportement des contrôleurs, les routes, les actions, les vues et d'autres aspects du framework. Les annotations incluent :

- **Routing** : définir les routes au-dessus des actions du contrôleur
- **Templating** : spécifier les templates Twig
- **Security** : déclarer les permissions requises
- **Cache** : spécifier les règles de mise en cache
- **Form Validation** : convertir et valider automatiquement les données de formulaire

### JMSSerializerBundle

Facilite la sérialisation et la désérialisation d'objets PHP dans divers formats (JSON, XML, YAML). Les annotations contrôlent le processus de sérialisation et gèrent les relations entre objets, permettant la conversion entre objets PHP et données sérialisées pour la communication avec des API externes.

## Conclusion

Les bundles permettent d'ajouter de nouvelles fonctionnalités Symfony via des packages tiers. Certains s'installent automatiquement ; d'autres nécessitent un ajout manuel en fonction des besoins du projet. Efficience IT sélectionne les bundles appropriés en fonction des besoins spécifiques de chaque projet.
